Daum Space.2 in Jeju Timelapse

장비
------
* Raspberry Pi
* GoPro Hero3 HD
* Canon PowerShot G6
* Raspberry Pi Camera Module

촬영기간

* 일출~일몰까지 GoPro 2대는 5분당 1장씩, G6와 RPi Camera는 15분당 1장씩 촬영
